# MusicBot

MusicBot é um chatbot com foco em tudo relacionado a musica, você pode utiliza-lo para consultas ou recomendações musicais, curiosidades sobre a musica ou banda e criação de playlists, ele utiliza principalmente a API do Spotify e tambem a API ReccoBeats para buscar informações musicais, analisar características das faixas, descobrir artistas semelhantes e playlists diretamente em sua conta do Spotify.

---

**Pré-requisitos**

- Python 3.11+
- Node.js 18+
- PostgreSQL
- React

---

**Instalação**

Primeiramente será necessário baixar os pré-requisitos, basta digitar os seguintes comandos no terminal do seu SO:

***Python:***

*Windows*
```
winget install Python.Python.3.11
```

*Ubuntu*
```
sudo apt update

sudo apt install python3.11 python3.11-venv python3-pip -y
```

***Node.js***

*Windows*
```
winget install OpenJS.NodeJS.LTS
```
*Ubuntu*
```
sudo apt update
sudo apt install nodejs npm -y
```
***React***

React não é instalado diretamente, é necessario o Node.js (que inclui o npm) e depois criar um projeto React, os comandos de criação de projeto são o mesmo independente do SO


*Ubuntu e Windows*
```
npx create-react-app musicbot
cd musicbot
npm start
``` 
***PostgreSQL***

*Windows*
```
winget install PostgreSQL.PostgreSQL
```

*Ubuntu*
```
sudo apt install postgresql postgresql-contrib -y
```
